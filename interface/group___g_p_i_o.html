int gpio_set_int ( unsigned int  pin,
void(*)(int)  isr,
char *  mode 
)

Set isr as the interrupt service routine (ISR) for the pin. Mode should be one of the strings "rising", "falling" or "both" to indicate which edge(s) the ISR is to be triggered on. The function isr is called whenever the edge specified occurs, receiving as argument the number of the pin which triggered the interrupt.

Parameters
pinPin number to attach interrupt to
isrInterrupt service routine to call
modeInterrupt mode
Returns
Returns 1 on success. Never fails
